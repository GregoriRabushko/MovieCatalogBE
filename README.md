
# Project Name

A Node.js/Express REST API application with PostgreSQL database support and Docker containerization.

## Features

- RESTful API built with Express.js
- PostgreSQL database integration with Knex.js query builder
- Database migrations and seeds support
- Docker and Docker Compose configuration
- CORS enabled for cross-origin requests
- Development mode with hot-reload using Nodemon

## Tech Stack

- **Runtime**: Node.js 25.1.0
- **Framework**: Express 5.1.0
- **Database**: PostgreSQL
- **Query Builder**: Knex.js 3.1.0
- **Other Libraries**:
    - cors 2.8.5
    - uuid 13.0.0
    - pg 8.16.3

## Prerequisites

- Node.js (v25.1.0 or higher)
- npm
- PostgreSQL (if running locally)
- Docker and Docker Compose (optional, for containerized setup)
